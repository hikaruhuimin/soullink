# SoulLink OG Image Specifications

## Required OG Images

### 1. Homepage OG Image (1200x630px)
**Recommended Content:**
- Cosmic/galaxy background with stars
- SoulLink logo prominently displayed
- Tagline: "Discover Your Cosmic Soulmate"
- Subtle mystical elements (cards, stars, moon)
- Clean, modern design with spiritual undertones
- Color palette: Deep purple, gold, white accents

### 2. Divination OG Image (1200x630px)
**Recommended Content:**
- Tarot cards or mystical symbols
- "AI Divination" headline
- Cosmic background
- "Explore Your Destiny" subtext

### 3. Zodiac Compatibility OG Image (1200x630px)
**Recommended Content:**
- Two zodiac signs (circle/constellation style)
- Heart symbol between them
- Compatibility percentage display
- "Find Your Cosmic Match" headline

### 4. Tarot Reading OG Image (1200x630px)
**Recommended Content:**
- Fan of tarot cards (3-5 visible)
- Mystical glow effect
- "Daily Tarot Reading" text
- Cosmic/mystical background

### 5. Social Media Banner (1500x500px)
**Recommended Content:**
- Horizontal banner format
- All three language versions side by side or stacked
- SoulLink branding
- Key features listed
- CTA button

## File Naming Convention
- `og-image-home-en.png` (English homepage)
- `og-image-home-zh.png` (Chinese homepage)
- `og-image-home-ja.png` (Japanese homepage)
- `og-image-divination-en.png`
- `og-image-tarot-en.png`
- `og-image-horoscope-en.png`
- `og-image-mbti-en.png`
- `social-banner.png`

## Technical Requirements
- Format: PNG (preferred) or JPG
- Minimum size: 1200x630px
- Maximum file size: 5MB
- sRGB color space
- Text should be minimum 30px (for readability)
